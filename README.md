## 启动设置信息
```yaml
# smartKey setting
icd:
  server-address: 127.0.0.1:7777
  generate-number: 100
```

### 注意事项
- 启动时，需要确保生成的钥匙信息都存在对应的环境中
- 确保每个钥匙设备都有绑定车和设备
